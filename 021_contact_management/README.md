# Contact Management System (CLI)

## Overview
This project is a command-line Contact Management System written in pure Python.
It allows users to add, view, delete, and search contacts with basic input validation.

The goal of this exercise is to practice object-oriented programming, input validation,
and building a menu-driven CLI application.

---

## Features
- Add multiple contacts
- Automatic ID generation
- Input validation for name, email, and phone number
- View all contacts
- Delete contacts by ID
- Search contacts across all fields
