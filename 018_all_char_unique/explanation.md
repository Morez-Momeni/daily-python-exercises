# Problem 18: Check if Two Strings are Rotations of Each Other

## Problem
Write a function to determine if a string has all unique characters (i.e., no
character is repeated)

**Examples:**
- `("abcde")` → `True` 
- `("a")` → `True`
- `("aabb")` → `False`
- `("admin")` → `True`
- `("hello")` → `False`
